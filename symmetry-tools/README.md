Symmetry-tools
==============

This project packages CE-Symm and related tools for easy deployment.

Tools can be run or downloaded from http://source.rcsb.org/jfatcatserver/symmetry.jsp.

Documentation
-------------

* [CE-Symm](docs/CeSymm.md)
* CensusCLI
